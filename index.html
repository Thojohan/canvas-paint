What is this?

This is a simple paint tool I made from scratch in HTML, CSS and vanilla JavaScript, as a practice exercise. The basic functionality is painting (mouse or touch), and selecting start/end values for color and line width, and change rate.

How to paint/draw

You draw by pressing and holding the left mouse button. Right mouse toggles the control bar. Default behaviour on right click is deactivated, so to access dev tools etc. you have to use the browser menu. On mobile/touch devices everything works the same except you can't toggle the control bar.

What do the sliders do?

It's pretty simple - the drawing tool uses the HSL (Hue Saturation Lightness) color spectre. You chose the start value and end value for all three, plus line width, and when you draw it will cycle forth and back through the values between your start/end points. The change rate determines how fast it cycles through. The rate is set slightly different for H/S/L/width values so the change isn't linear.

What does the "Stroke Reset" button do?

Default behaviour is to start each brush stroke from the start value set by the sliders. You can use this button to override this behaviour. Meaning that if you stop drawing/painting the next line you start will start from the color and line width that finished the last line instead.
